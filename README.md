# Setup

## Dependencies

### uv
python dependency and virtual environment manager.

https://docs.astral.sh/uv/getting-started/installation/

### ollama
open source LLMs.

see https://ollama.com/download

## Initialization

from within the project folder, run:

```bash
uv sync
```

# Running the project

To start the server, run:

```bash
uv run app.py
```
